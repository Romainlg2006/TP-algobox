\# Projet AlgoBox - TP GitHub



\## Description

Ce projet contient un algorithme développé avec AlgoBox dans le cadre du TP sur la collaboration avec Git/GitHub.



\## Contenu

\- `src/EXOTP3.alg` : Fichier principal de l'algorithme AlgoBox



\## Comment ouvrir et exécuter



\### Prérequis

\- AlgoBox installé (téléchargeable sur http://www.xm1math.net/algobox/)



\### Instructions

1\. Ouvrir AlgoBox

2\. Aller dans `Fichier > Ouvrir`

3\. Sélectionner le fichier `src/EXOTP3.alg`

4\. Cliquer sur "Exécuter" pour lancer l'algorithme
